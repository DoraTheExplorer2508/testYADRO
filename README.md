# Тестовое задание
## Запуск программы (MinGW)
Склонировать репозиторий

```
git clone https://github.com/DoraTheExplorer2508/testYadro
```

Перейти в директорию

```
cd testYadro
```

Скомпилировать файлы

```
g++ -o main Main.cpp CompClub.cpp
```

Запуск

```
./main file1
```
